Как сгенерировать решение на VS15:
1. В системе должен быть установлен OpenCV
2. В командной строке: 
mkdir build
cd build
cmake -G "Visual Studio 15 2017 Win64" ".."
cd ..
(для удобства продублировано в generate.bat)
3. В папке build будет лежать проект

Собраное приложение (x64) лежит в папке bin
Изображения, карты глубины и результаты работы лежат в папке data

Как пользоваться:
test_task.exe -in_rgb 3c.png -in_dm 3b.bin -out 3out.png

Порядок аргументов значения не имеет
-in_rgb [file] - исходная картинка с цветами
-in_dm [file] - карта глубины
-out [file] - картинка, в которую сохраняется проекция
